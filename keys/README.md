# ICICI Payment Gateway Keys

Place your cryptographic keys here:

1. icici_public_key.pem - ICICI Bank's public key (provided by ICICI)
2. merchant_private_key.pem - Your merchant private key (generated by you)
3. merchant_public_key.pem - Your merchant public key (generated by you, send to ICICI)

## Generating Keys

To generate RSA 4096-bit keys for merchant:

```bash
# Generate private key
openssl genrsa -out merchant_private_key.pem 4096

# Generate public key from private key
openssl rsa -in merchant_private_key.pem -pubout -out merchant_public_key.pem
```

Send merchant_public_key.pem to ICICI Bank during onboarding.

## Security
- Never commit private keys to git
- Keys directory is already in .gitignore
- Store backup of private key securely
